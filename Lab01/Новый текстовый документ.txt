Метод Симпсона - это численный метод для вычисления приближенного значения определенного интеграла 
функции на заданном интервале. Этот метод основан на аппроксимации функции на каждом интервале 
квадратичной кривой, проходящей через три точки: левый конец интервала, правый конец интервала и 
середина интервала. При использовании метода Симпсона интервал интегрирования разбивается на четное 
число подинтервалов, на каждом из которых применяется формула Симпсона. Общее значение интеграла 
вычисляется путем суммирования значений интегралов на каждом из подинтервалов.

Метод трапеций - это один из численных методов интегрирования функций, который основывается на 
аппроксимации подынтегральной функции на каждом отрезке интегрирования линейной функцией. Данный 
метод получил свое название из-за того, что подынтегральная функция на каждом отрезке заменяется 
на трапецию, верхняя грань которой является значением функции в правом конце отрезка, а нижняя - 
в левом конце.

Идея метода трапеций заключается в том, чтобы разбить область интегрирования на небольшие отрезки 
и аппроксимировать подынтегральную функцию на каждом отрезке линейной функцией, что позволяет вычислить 
значение определенного интеграла численно.

Метод прямоугольников (или метод прямоугольников простой) - это численный метод, 
используемый для приближенного вычисления определенного интеграла функции.

Суть метода заключается в том, что под интегралом функция заменяется на некоторое простое 
выражение, например, на значение функции в середине каждого интервала разбиения интеграла. 
Затем значения этого выражения на каждом интервале умножаются на ширину интервала и складываются, 
чтобы получить приближенное значение определенного интеграла.

Если использовать прямоугольники, которые вычисляются как значение функции на левой или правой 
границе интервала, то соответствующий метод называется методом левых или правых прямоугольников. 
Если использовать среднее значение функции на интервале, то метод называется методом средних 
прямоугольников.
